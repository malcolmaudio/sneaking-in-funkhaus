# sneaking in Funkhaus

## Overview
sneaking in Funkhaus is a multi-algorithm algorithmic reverb plugin. It builds upon classic reverberation topologies by providing modern shaping tools like tone control, saturation, and dual-chorus modulation.

## Algorithms (Spaces)
1. **Cathedral**: Huge size, echoey, brighter damping.
2. **Hall**: Standard room model.
3. **Plate**: Bright, dense, and short.
4. **Chamber**: Smaller, darker sound.
5. **Spring**: Metallic, minimal damping, narrow stereo width.
6. **Shimmer**: Massive, bright, naturally paired with chorus.
7. **Cone**: Focused and directional.
8. **Non-Linear**: Artificial, gated response.

## Character Modes
- **Clean**: Transparent algorithmic reverb.
- **Glimmer**: Dual chorus applied to the reverberant signal for a lush, wide stereo image.
- **Grit**: Saturation and bitcrushing for lo-fi textures.

## Key Controls
- Pre-Delay, Decay, Size, Width, Damping
- High Pass Filter
- Clean, Glimmer, Grit toggle
